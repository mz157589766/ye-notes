大多时候都是在配置环境变量的时候，配置错误导致系统自身的环境变量被覆盖，从而任何一个命令都是command not found。

这个时候，可以先手动临时声明进行解救：

```sh
export PATH=$PATH:/bin
```

此时会发现命令暂时可用了，然后再把刚刚配置错误的变量修复即可！
