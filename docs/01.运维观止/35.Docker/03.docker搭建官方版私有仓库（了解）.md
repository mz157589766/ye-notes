## 1，直接通过如下命令启动私有仓库。



```sh
docker run -d --name registry -v /opt/registry:/var/lib/registry -p 5000:5000 --restart=always registry
```



## 2，通过配置，连接私服。



```sh
vim /etc/docker/daemon.json
{
    "insecure-registries": ["192.168.106.5:5000"]
}
 
systemctl restart docker
```
