这个配置重点其实还是一个静态代理，通过root或者alias都是可以的，两种方式分别如下：

```sh
# Sitemap.xml
location /sitemap.xml {
    alias /venvs/mobilepro/src/mobilepro/static/maps/map.xml;
}
```

或者：

```sh
# Sitemap.xml
location /sitemap.xml {
    root /venvs/mobilepro/src/mobilepro/static/maps;
}
```
