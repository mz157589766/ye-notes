- 将副本重置为0，如果数据重要性不是特别大，可以采用这个配置，以释放es写的压力。

```json
PUT /test-index*/_settings
{
   "settings" : {
      "number_of_replicas" : 0
   }
}
```

- 设置分片在每个节点上都有

```json
PUT /test-index/_settings
{
"index.routing.allocation.total_shards_per_node":1
}
```
